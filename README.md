# UnCAPTCHAble 

UnCATPCHAble is a brand new CAPTCHA that makes life really hard for AIs while still being easy to solve for humans.

# Requirements
- Node.js


# Installation
## Clone
```
git clone git@github.com:ivanarena/uncaptchable.git
```

## Run API 
```
cd uncaptchable
node .
```
## Run client
```
cd client
npm start
```
